# Historical Archive

The files in this directory are historical Anchor examples, IDL snapshots, and generated types kept for reference only.

They may contain older endpoint hosts such as `oracle-dev.txodds.com`, older activation flows, and legacy instructions such as `subscribe_with_token`. Do not use this directory as the current TxLINE integration path.

Use the current documentation instead:

- [Quickstart](https://txline.txodds.com/documentation/quickstart)
- [Program Addresses](https://txline.txodds.com/documentation/programs/addresses)
- [IDL & Types (Mainnet)](https://txline.txodds.com/documentation/programs/mainnet)
- [IDL & Types (Devnet)](https://txline.txodds.com/documentation/programs/devnet)
