TxLINE odds are powered by Stable Price, TxODDS' consensus pricing engine. Merkle roots are published to Solana, allowing supported proof-backed price records to be checked against the matching on-chain commitment.

Stable Price aggregates lines across global operators, including sharp books absent from standard Western feeds. The engine runs defensive logic to filter outliers, stale lines, and bad data before it ever reaches your application.

## Access & Onboarding

Start a guest session, submit the matching on-chain subscription, then activate an API token. Free World Cup tiers require no TxL payment; paid tiers use TxL. Keep the Solana network, program ID, guest JWT host, and API host aligned.
