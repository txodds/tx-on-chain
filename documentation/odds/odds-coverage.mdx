## Free-Tier Stable Price Odds

Since November 7, 2025, the following endpoints provide free access to de-margined odds with 60-second delay:

- `GET /guest/odds/snapshot/{fixtureId}`
- `GET /guest/odds/stream`

These endpoints only require a JWT token (no subscription needed) and cover the following competitions:
