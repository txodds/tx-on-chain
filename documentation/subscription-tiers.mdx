## Overview

TxLINE offers two types of data access subscriptions, each available in flexible league bundles. All subscriptions are billed weekly.

## 60-Second On-Chain Updates

Get access to 60-second Merkle Root updates published on-chain for your selected bundle of soccer leagues.

### Pricing

All prices are per 7-day subscription period:

| Bundle | TxL Tokens |
|--------|------------|
| 10 Leagues | 500,000 TxL |
| 25 Leagues | 750,000 TxL |
| 50 Leagues | 1,000,000 TxL |
| 100 Leagues | 1,250,000 TxL |
| All Leagues | 2,500,000 TxL |

## Real-Time Updates via Fusion

Access real-time stable price updates delivered off-chain through Fusion for maximum speed and reliability.

### Coverage

- **Sport**: Soccer
- **Odds Types**: 1x2, Over/Under, Asian Handicap
- **Markets**: Full Time, 1st Half, 2nd Half
- **Leagues**: Average of 562 active leagues per month (ranging from 237 in January to 716 in October)

### Pricing

All prices are per 7-day subscription period:

| Bundle | TxL Tokens |
|--------|------------|
| 10 Leagues | 5,000,000 TxL |
| 25 Leagues | 7,500,000 TxL |
| 50 Leagues | 10,000,000 TxL |
| 100 Leagues | 12,500,000 TxL |
| All Leagues | 25,000,000 TxL |

## Payment

Subscriptions are purchased using TxL tokens and automatically renew weekly unless cancelled.
